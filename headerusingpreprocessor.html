HEADER USING PREPROCESSOR

This module allows for EUPHORIA code to include C headers right at the top of a EUPHORIA program. Translation is done on the fly. Once translated the resulting dot-e files can be copied to another location.

Supported Commands in the EUPHORIA files

include headerfile.h

This includes most of a header's constants and routines for use with EUPHORIA. Until we can parse arbitrary C expressions macro constants will never be 100% converted but most are.

with c_errors

This makes the preprocessor enable or disable errors. When enabled it will die when it encouters something it cannot handle.

global constant constant_name = constant_value

This creates a global symbol and this global symbol will also be visible to header files you include after this declaration.

Supported C commands are macro constants.

#define new_symbol definition

static const new_symbol = definition

Currently single token literals (#defined values or numeric literals) are supported.
